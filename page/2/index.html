未命名

发表于 2017-05-11

自动居中一列布局

  • 处于标准文档流中
  • 块级元素
  • 设置宽度和 margin：0 auto即可

    设置浮动或绝对定位则会使自动居中失效，因为其会脱离文档流。

浮动的横向两(多)列布局

  • float使纵向排列的块级元素横向排列
  • 通过margin设置列之间的间距

绝对定位的横向两列布局

常用于一列定宽，另一列自适应

  • relative 父元素相对定位
  • absolute 子元素绝对定位，定宽元素设置宽度，自适应元素不设置宽度

三栏中间自适应布局

两列定宽，中间自适应

方法一：

  • 三栏父元素设置相对定位
  • 剩下三栏设置绝对定位，定宽设置宽度，自适应不设置宽度。

方法二：

  • 三栏父元素设置两边padding，留出两栏定宽部分

或者

后者学习外边距负值的意义即可，实际开发中不使用（浏览器缩小到一定范围，左右两边栏会被挤到下一行）

  • 利用绝对定位，设置两栏定宽部分

    方法三：

  • 中间一栏设置margin，留出两栏定宽部分
  • 利用绝对定位，设置两栏定宽部分

jquery-ajax

发表于 2017-05-08 | 分类于 jQuery

1. load()

从服务器加载数据，并把返回的数据放入被选元素中
$(selector).load(URL,data,callback)

  • URL：希望加载的URL，必需参数
  • data：规定与请求一同发送的查询字符串键/值对集合，可选参数
  • callback ： load() 方法完成后所执行的函数名称

2. get() 和 post()

get():通过 HTTP GET 请求从服务器上请求数据

$.get(URL,callback)

post(): HTTP POST 请求从服务器上请求数据

$.post(URL,callback)
