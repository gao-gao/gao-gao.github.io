SEO优化简介

发表于 2017-08-02 | 分类于 SEO
一些常见的优化方式

网站结构布局优化（扁平化结构）

控制首页链接数量

  • 中小型网站100个左右以内，链接的性质包括页面导航，底部导航，文字链接

    扁平化的目录层次

  • 三层

导航SEO优化

  • 主导航，副导航，分类导航，层级导航
  • 层级导航（让用户了解当前所处位置和网站结构层次）

    控制页面大小和加载速度

    细节

  • 分页，下拉框显示页数进行访问，方便爬虫读取

网页代码优化

title标题

meta keyword 关键词

meta description 网页描述

body中代码

  • 使用语义化标签，样式效果用css实现

  • a 标签，加上title说明；外部链接加入rel="notfollow"提醒爬虫不用查看

  • h1标签，按标题重要性决定用几号标题而不是按照大小

  • table标签，使用 caption

  • img 标签使用alt和title属性

  • br标签放在p标签内；文本内不要加入&nbsp；

  • strong em 权重高于 b i

注意事项

  • 重要内容HTML代码放在最前面（css处理位置）

  • 重要内容不用js输出，爬虫不会读取js和flash

  • 尽量少使用iframe框架(不会读取内部内容)

  • 谨慎使用display:none(会过滤掉此部分代码)

  • 不断精简代码
