svg基础知识

发表于 2017-11-08 | 分类于 html5&css3
svg的优势以及通过svg绘制形状、路径、文本，添加滤镜，绘制动画等

SVG可缩放矢量图形

优势

  • 便于读取和修改：可被多种工具读取和修改，图像中文本是可选、可搜索（制作地图）的
  • 小：相对于jpeg和gif图像，尺寸更小且可压缩性更强
  • 可伸缩：在任何分辨率下被高质量地打印，在图像质量不下降的情况下被放大
  • 开放标准：是纯粹的xml,可与java技术一起运行，兼容其他标准如DOM
  • 主流浏览器均可直接打开并查看SVG文件

SVG视口 viewport

SVG可见区域的大小，或者理解为画布大小<svg width="" height=""></svg>

单位

svg默认单位是px，不注明单位时就为px

  • em：相对父元素的字体大小
  • ex：相对小写字母x的高度
  • px：相对于屏幕分辨率，一个点或者72分之一英寸
  • in：英寸
  • cm：厘米
  • mm：毫米
  • pt：72分之一英寸
  • pc：12分之一点
  • %：相对于父元素的百分比

视区盒子 viewBox

被选中的要在视口viewport中全屏显示的区域viewBox="x,y,width,height"

preserveAspectRatio

让viewBox保持宽高比[defer]<align-x><meetOrSlice>，由两个值构成，一个值确定对齐方式，一个值确定缩放方式

defer

可选值，仅在image元素上应用此属性时才会使用

align由x,y两个方向上的值确定(驼峰命名法拼接 )

控制viewBox的在viewport中占的位置(b:viewBox p:viewport)

  • xMin：b的最小X值对齐p的左边部
  • xMid：b的X轴中点对齐p的X轴中点
  • xMax：b的最大X值对齐p的右边部
  • yMin,yMid,yMax

    meetOrSlice参数

    控制viewBox缩放的方式
  • meet：保持宽高比并将viewBox缩放为适合viewport的大小，以最小的宽高为基准缩放
  • slice：保持宽高比并将所有不在viewport中的viewBox剪裁掉，尽量填满viewport，以最大的宽高为基准缩放
  • none：不保存宽高比，缩放图像适合整个viewbox，可能会发生图像变形

坐标系统

世界坐标系统

以左上角为(0,0)的计算机图形学中的世界坐标系

最初坐标系统

<svg></svg>即画布，viewport部分

用户坐标系统

<svg><rect></svg> 即rect部分内定义的坐标系统

转换坐标系统

通过transform进行变换，如缩放、移动、倾斜、旋转，相对于画布左上角，自带偏移且不支持3D，属性值不能加单位默认px <rect transform="translate(100)">

  • translate(|),ty值为可选值，省略默认为0，相对于svg的左上角
  • rotate([]),angle为旋转角度，cx和cy设置旋转中心点
  • scale([]),沿x/y轴的缩放值，用来延长或者拉伸元素
  • skewX(angle) skewY(angle)
    渲染机制是，先将坐标系统偏移(缩放)，再渲染元素

svg2

把类似transform-origin和3Dtransform引入，持观望态度

svg形状

矩形

<rect x="" y="" rx="" ry="" width="" height=""/>

  • x,y定位rect左上角位置，默认为0
  • rx,ry定义圆角半径，默认为0

圆形

<circle cx="" cy="" r=""/>

  • cx,xy 圆心的位置
  • r 圆的半径

椭圆

<ellipse cx="" cy="" rx="" ry=""/>

  • cx,cy 椭圆的中心点位置
  • rx,ry 椭圆的x,y半径

线

<line x1="" y1="" x2="" y2="">

  • x1,y1 起点坐标
  • x2,y2 终点坐标

折线

<polyline points="x1 y1,x2 y2,x3 y3...,xn yn">

  • xn yn 点集数列，分别表示折线各点的坐标。每个数字用空白符、逗号、终止命令或换行符分隔开
  • 第一个点和最后一个点相同时，闭合点没有完美融合

多边形

<polygon points="x1 y1,x2 y2,x3 y3...,xn yn"/>

  • 同上折线 不过路径绘制完后自动闭合图形，形成一个闭合的多边形

路径

通过d定义，d是一个点集数列和路径信息，值是一个命令+参数的序列。不仅可以创建以上基本形状，还可以创建更多形状<path d="">

画笔命令

大写字母为绝对命令，小写字母为相对命令

  • M x y,将画笔移动到(x,y)处
  • m dx dy, 将画笔向右移动x,向下移动y
    直线命令
  • L x y,在之前位置和新位置之间画一条线段
  • l dx dy,在l命令之前的点到移(dx,dy)那个点之间画一条线段
  • H x（h dx),绘制水平线到定义位置
  • V y（v dy）,绘制垂直线到定义位置
  • Z（z）,闭合路径命令，连接当前点和起点(直线)
曲线命令

  • 三次贝塞尔曲线
    C x1 y1,x2 y2,x y或者c dx1 dy1,dx2 dy2,dx dy,(x,y)表示曲线的终点,起点由M命令确定,(x1,y1)起点控制点,(x2,y2)终点控制点。描述曲线上各个点的斜率
    S x2 y2,x y或者s dx2 dy2,dx dy将若干个贝塞尔曲线连起来，创建出一条长而平滑的曲线，起始点默认为前面的命令的对称点。

  • 二次赛贝尔曲线
    Q x1 y1, x y或者q dx1 dy1,x y,(x,y)表示曲线的终点，(x1,y1)控制点既控制起点又控制终点
    T x y或者t x y类似于S命令，连接赛贝尔曲线

  • 弧形
    A rx ry x-axis-rotation laege-arc-flag sweep-flag x y,xy椭圆两轴半径，椭圆的旋转角度，弧线是大角度弧1还是小角度弧0,弧线方向0逆时针1顺时针,xy指定弧形的终点

svg文本

可直接复制粘贴

字符，字形和字体

字形和字体被称为字体数据

  • 字符：要展示的字母的数字表示形式，由Unicode定义
  • 字形：字符的可视化表示形式,衬线非衬线体等
  • 字体：字形的集合

    EM框

    (0,0)点是在左下端
  • Ascent:从字体的(0,0)点到EM box顶部的距离
  • Desent:从字体的(0,0)点到EM box底部的距离
  • baseline table:基线，设计空间坐标系统中的基线的位置

text元素

<text></text>,可以放任何文字

  • 设置字体属性：font-family,font-style,font-weight,font-variant,font-stretch,font-size,font-size-adjust,kerning,letter-spacing,word-spacing,text-decoration

文本的位置

<text x="" y=""></text>设置在视图中展示文本的位置

  • x=0 y=0 时文本并不在svg框中，因为文本em框的(0,0)点在文本左下方
    偏移
  • x=”x1 x2 x3 …xn” y=”y1 y2 y3…yn”根据坐标指定位置，后续没有指定数值的默认为最后一个值
  • dx=”x1 x2 x3 …xn” dy=”y1 y2 y3…yn”根据坐标对文本进行偏移，后续没有指定的为最后一个值
    旋转
  • rotate="deg1 deg2 deg3...degn"，根据给定角度对字符进行匹配旋转，如果只给一个值，则匹配所有
  • transform="rotate(deg)",转换坐标系统对整个文本行进行旋转

    文本的长度

  • textLength="widthValue"平均计算文本键的间距
  • lengthAdjust="spacing"spacing为默认值,spacingAndGlyphs将文本拉宽

tspan元素

<text><tspan></tspan></text>标记大块文本的子部分,可以给其一个特殊样式

textPath元素

  • <path id="path" d="" stroke="" fill="">设置路径
  • <text><textPath xlink=""></textPath></text>利用xlink:href属性把字符对齐到路径，让字体环绕路径，顺着路径走

填充和边框

上色

  • fill 设置对象内部的颜色
  • fill-opacity 控制填充色的不透明度
  • stroke 设置绘制对象的线条的颜色
  • stroke 控制描边的不透明度

    描边

  • stroke-width 描边的宽度（以路径为中心线绘制）
  • stroke-linecap 边框终点的形状，butt(直线结束),square(稍超实际路径),round(圆的)
  • stroke-linejoin 两条描边线段之间的连接方式，miter尖角,bevel斜接切边,round圆角平滑
  • stroke-dasharray 将虚线类型应用在描边上，添加长度定义复杂的虚线模式(逗号分隔)

使用CSS

  • 使用style标签
  • 外链css<?xml-stylesheet type="text/css" href="style.css"?>

svg动画

SMIL(同步多媒体集成语言) animation概览

  • 动画元素的数值属性(x,y…)
  • 动画属性变换(平移或旋转)
  • 动画颜色属性
  • 沿着运动路径运动

    效果

  • <set>设置，在特定时间之后修改某个属性值
  • <animate>基础动画元素，实现单属性动画过渡效果
  • <animateTransform>实现transform变换动画效果
  • <animateMotion>让SVG各种图形沿着特定的path路径运动
  • 组合动画，多个动画组合于一个元素，构成复杂动画

    参数详解

  • attributeName=””，要变化的元素属性名称
  • attributeType=”css/xml/auto”，表明attributeName属性值的列表
  • from,动画的起始值
  • to,动画的结束值(绝对值)
  • by,动画的相对变化值
  • values,用分号分隔的一个或多个值，可以看成是动画的多个关键值点
begin，end

begin,动画开始的时间;end,动画结束的时间

  • begin=”value1;value2” 1时动画走一下，2时动画再走一下
  • offset-value:偏移值
  • syncbase-value:基于同步确定值
  • event-value:与事件相关联的值
  • repeat-value:重复多少次之后执行
  • accessKey-value:定义快捷键，按下某个按键动画开始
  • wallclock-sync-value:真实世界的时钟事件定义
  • indefinite:无限等待
    dur
    动画持续时间’indefinite’为动画时间无限
    calcMode,keyTimes,keySplines
    控制动画快慢曲线
  • discrete:from值直接跳到to值
  • linear:速率一致
  • paced:通过插值让动画变化步调平稳均匀
  • spline:插值定义贝塞尔曲线
    #####repeatCount,repeatDur
    前者表示动画执行次数，后者定义重复动画的总时间
    fill
    表示动画间隙的填充方式
  • remove 默认值 动画结束直接回到开始的地方
  • freeze 动画结束后保持动画结束之后的状态
    accumulate=”sum”
    动画结束时候的位置作为下次动画的起始位置

动画的暂停与播放

在js中控制，直接在svg上运行，停止时都停止

  • 暂停 svg.pauseAnimations()
  • 播放 svg.unpauseAnimations()

svg嵌入html

使用svg文件

  • <embed> 将svg文件以媒体形式插入html

  • <object> 将svg文件以媒体对象形式插入html

  • <iframe> 将svg文件以网页文件形式插入html

  • <image> 将svg文件以图形图像文件形式插入html

嵌入svg结构

使用svg标签直接将svg代码插入到html中
