css3新属性

发表于 2017-11-25 | 分类于 html5&css3
word-break和word-wrap的区别，tex-overflow文本溢出时的省略效果，字体规则@font-face，columns属性分列相关参数，用户界面中的一些属性

css3文本效果

文本阴影 text-shadow

模糊半径设置成负数时没有任何模糊效果

  • text-shadow:x y blur color

换行 word-break

normal,break-all,keep-all断开单词进行换行

  • normal 使用浏览器默认的换行规则
  • break-all 允许在单词内换行，不会浪费空间
  • keep-all 只能在半角空格或连字符处换行

单词换行 word-wrap

允许长单词或 URL 地址换行到下一行（一行还是放不下时断开去下一行），会浪费空间

  • word-wrap:normal | break-word

新文本属性

text-align-last规定如何对齐文本的最后一行

只有IE支持

  • text-align-last:auto|left|right|center|justify|start|end|initial|inherit
    text-emphasis向元素的文本应用重点标记以及重点标记的前景色
  • text-emphasis:style color
text-justify
  • text-justify:auto|inter-word|inter-ideograph|inter-cluster|distribute|lashida|trim
text-overflow文本溢出包含元素时发生的事情
  • text-overflow:clip|ellipsis|string

字体

能用常规字体最好用常规字体，使用特殊字体渲染时用户需要从服务器下载字体，耗费时间，影响网站性能。

@font-face规则

name

自定义字体名称

source

字体路径，绝对路径或者相对路径

format

定义字体格式，推荐通用模板

  • .ttf Windows和mac最常见的字体，RAW（写入式）格式，不为网站做优化，通常用于手机端。
  • .otf 原始的字体格式，其内置在ttf的基础下，不兼容IE
  • .woff web字体中的最佳格式，开放的ttf和otf的压缩版本，同时支持元数据包的分离，不兼容手机端iPhone
  • .eot IE专用字体
  • .svg 基于SVG字体渲染的一种格式，IE火狐不兼容
weight

定义字体粗细，如粗体

style

定义字体样式，如斜体

多列

columns 列数和宽度

columns:<width>||<count>总宽度不够时优先考虑宽度，宽度足够时优先考虑列数

  • width:length|auto(根据列数自定分配宽度)不可以使用负值
  • count:不设置列数时根据列宽自动控制列数

column-gap 列与列之间的间隙

column-gap：length|normal默认是normal

  • normal:和字体大小相同

column-rule 列与列之间的边框

column-rule:width|style|color

  • width:边框厚度length|thin|medium|thick
  • style:边框样式none|hidden|dotted|dashed|solid|double|groove|ridge|inset|outset
  • color:边框颜色

column-span 元素是否横跨所有列

column-span:none|all

  • none:不跨列
  • all:横跨所有列

column-fill 所有列的高度是否统一

column-fill:auto|balance主流浏览器都不兼容

  • auto:列高度自适应内容
  • balance:所有列的高度都以最高的一列统一

column-break 判断对象是否断行

column-break-before 在对象之前是否断行

column-break-before:auto|always|avoid

  • auto:既不强迫也不禁止在元素之前断行并产生新列
  • always:总是在元素之前断行并产生新列
  • avoid:避免在元素之前断行并产生新列
    column-break-after 在对象之后是否断行
    同上
    column-break-inside 在对象内部是否断行
    column-break-inside:auto|avoid

用户界面 appearance

appearance:none|button|button-bevel...属性值有110个，所有主流浏览器都不支持appearance属性

text-overflow

在块容器<’overflow’>为非 visible 时，定义内联内容溢出其块容器是否截断。要结合 white-space:nowrap;和 overflow:hidden;才起作用。

  • clip:当内联内容溢出块容器时，将溢出部分裁切掉
  • ellipsis:当内联内容溢出块容器时，将溢出部分替换为…

outline 线条轮廓

outline:width|style|color画在border外面，不占据布局空间，不影响元素尺寸

outline-width 线条轮廓的宽度

length|thin|medium|thick

outline-color 线条轮廓的颜色

color|invert

  • invert:使用背景色的反色(仅IE和Opera有效)
    outline-style 线条轮廓的样式
    none|dotted|dashed|solid|double|groove|ridge|inset|outset

    outline-offset 线条轮廓偏移容器的值

  • length:长度值定义轮廓偏移，允许负值

nav-index 导航顺序

auto|number即将被w3c废除

  • auto:元素的导航焦点顺序由客户端自动分配
  • number:用浮点数来定义元素的导航焦点顺序，若某元素的值等于1则意味着最先被导航，值相同时按照文档的先后熟顺序进行导航

    nav-up 对象的导航方向

    auto|id|[current|root|target-name]
  • auto:默认顺序
  • id:被导航元素的id
  • target-name:框架目标页面间的元素焦点导航
    于此相同用法的还有nav-down,nav-left,nav-right

cursor 光标形状

zoom 对象的缩放比例

参数

normal|number|percentage

  • normal 适用对象的实际尺寸
  • number 用浮点数来定义缩放比例，不允许负值
  • percentage 用百分比来定义缩放比例，不允许负值
    和scale的区别
  • scale: 以设置点为中心缩放，变换不会对布局产生影响
  • zoom: 以左上角为几点做缩放，先缩放后布局，可能影响到盒子的计算

    box-sizing 对象的盒模型组成模式

    content-box|border-box
  • content-box 宽高不包括padding和border
  • border-box 宽高包括padding和border

    resize 对象的区域是否允许用户缩放，调节元素尺寸大小

    需要设置overflow(auto hidden 或 scroll)，属性才会生效none|both|horizontal|vertical
  • none 不允许用户调整原宿大小
  • both 用户可以调节元素的宽度和高度
  • horizontal 用户可以调节元素的宽度
  • vertical 用户可以调节元素的高度

    ime-mode 是否允许用户激活输入中文、韩文、日文等的输入法（IME:input method editor）状态

    auto|normal|active|inactive|disable对于有焦点的控件（如输入框），用户不可以激活IME【待弃用】
  • auto 不影响IME的状态
  • normal 正常的IME状态
  • active 指定所有使用IME输入的字符
  • inactive 指定所有不使用IME输入的字符
  • disable 完全禁用IME，

    user-select 是否允许用户选中文本

    none|text|all|element
  • none 文本不能被选择
  • text 可以选择文本
  • all 当所有内容作为一个整体时可以被选择
  • element 可以选择文本 但选择范围受元素边界的约束

    pointer-events 在何时成为属性事件的target

    auto|none|visiblepainted|visiblefill|visiblestroke|visible|painted|fill|stroke|all
  • auto 与pointer-events属性未指定时的表现效果相同，在svg内容上与visiblepainted值相同（默认响应）
  • none 元素永远不会成为鼠标事件的target（不会响应）
  • 其他值只能应用在svg上
