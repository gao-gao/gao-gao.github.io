用css画汉克狗

发表于 2017-11-29 | 分类于 html5&css3
通过画一个动态的小狗狗加深对CSS的理解

汉克狗效果图

搞清楚汉克狗由哪几个部分组成

参考案例，可以在控制台中对狗狗各个部分的left和top值进行增减，更直观地感受汉克狗的各个部位

脸

效果图中脸的形状是不规则的，既不是圆，也不是椭圆，更像是多个图形堆叠起来的，如下图

所以在实现的过程中要将其分为三个部分，逐一实现，然后通过定位叠加到一起。

实现很简单，设置宽高，然后通过border-radius对div框进行圆角的设置。在实现的时候，无法对各个部位进行精确的定位以及大小、圆角值的精确计算时，可以先随意设置一个值，然后通过控制台进行调试，达到和效果图最相似的效果，然后修改源码中的数值。因为这仅仅是一次趣味性的实践，并没有严格的设计图去规定尺寸之类的，所以不必太过强迫症。

border-radius

在这里要强调一下border-radius的用法，通常用法是四个值，左上，右上，右下，左下这四个方向上的圆角的半径。但是这一个案例中，圆角通常都是椭圆，所以此处要同时设置x方向和y方向上的半径。即如下，/前面的四个值是四个方向上的x半径，/后面的四个值是四个方向上的y半径

耳朵

左耳和右耳是对称的，所以两只耳朵的样式基本一致。可以使用对称然后定位到左右耳相应的位置（下面的眼睛、手腿、脚都是采用这个方法）。但是此处为了加深一下border-radius画不规则圆框这个知识点，我选择了先画左耳，在左边定位，再画右耳，在右边定位的方法

此处需要注意的是，由于我不是将耳朵对称到右边，所以，左右耳的border-radius的各个方向设置值是相对称的，同样rotate旋转的方向也是相反的。

眼睛

眼睛部分细节比较多，比如右眼的蓝色胎记，左眼眼眶的阴影，眼球的层次，眼睛的上下睫毛等。

  • 蓝色胎记：创建新的div.birthmark

通过设置其层级z-index，将其下移，让其位于前面绘制的眼眶下面

  • 眼球的层次
    由三个部分组成，黄色圆，黑色圆，白色小圆，将其叠加，插到眼眶下一层（因为后面实现眼睛的移动时，瞳孔需要被眼眶遮住）。此处用过伪类的引入避免添加过多标签。

身体

由手、腿、脚、躯体、肚腩和手内侧的阴影遮盖组成

  • 手臂阴影和肚腩阴影：用过div遮盖实现
  • 腿内侧阴影：通过背景颜色实现background: linear-gradient(-90deg , var(--gray) 0%, var(--bg) 20%, var(--bg) 100%);

动画实现

一共有四处动画：耳朵的摇摆、瞳孔的移动、鼻孔的收缩、舌头的伸收

  • 耳朵的摇摆：rotate的角度的变化（由于两边耳朵不是通过对称实现的，所以需要单独对两个的动画进行设置）

  • 瞳孔的移动：left值的变化

  • 鼻孔的收缩：height值的变化
  • 舌头的伸收：height值的变化
