bootstrap栅格式布局

发表于 2017-06-02 | 分类于 bootstrap
bootstrap中网格布局的原理以及使用

实现原理

通过定义容器大小，平分12份，载调整内外边距，最后结合媒体查询，制作出了强大的响应式网格系统。 使用时可以根据实际情况重新编译LESS（或Sass）源码来修改12这个数值。

工作原理

栅格系统通过类container来盛装行，通过类row来盛装列（即每一个格子），通过类col-lg-*,col-md-*,col-sm-*,col-xs-*来设置每个格子所占的宽度,*号所代表的数字为占12分宽度中的数量，数字越大，占比越大

  • container类设置整个栅格的宽度，内补
  • row类设置整个行的样式

媒体查询

因为bootstrap是移动设备优先，默认为移动设备，所以不用对超小屏幕即768px以下的进行查询单独设置

媒体查询包含最大宽度，将css的影响限制在更小范围内

流式布局容器

将.container改为.container-fluid，即将固定宽度的栅格布局转换成宽度为100%宽度的布局

响应式列重置

列在分界点清除响应

  • 联合使用clearfix和响应式工具类

重置偏移，后推或者前拉这个类

  • .col-md-offset-*将列向右侧偏移
    为当前元素增加了左侧的边距（margin）

嵌套列

  • 通过一个新的row类嵌套在栅格里面

列排序

  • .col-md-push-*和.col-md-pull-*改变列的顺序
    push往后推，pull往前拉
