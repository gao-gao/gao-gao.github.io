html5文档类型

meta标签

  • viewport 元数据标签
    <meta name="viewport" content="width=device-width, initial-scale=1">
    width=device-width : 设置viewport的宽度为设备宽
    intial-scale ：设置初始缩放比例
    user-scalable=no ：在移动设备浏览器上设置可以禁用其缩放功能，此时用户只能通过滚动查看内容

全局样式

保留部分浏览器的基础样式，解决部分潜在的问题，提升细节体验：

  • 移除body的margin声明
  • 设置body的背景色为白色
  • 为排版设置了基本的字体、字号和行高
  • 设置全局链接颜色，且当链接处于悬浮“:hover”状态时才会显示下划线样式
