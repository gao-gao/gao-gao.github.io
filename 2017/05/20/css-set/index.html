页面中的标签的一些默认CSS设置介绍

排版

标题

和HTML中一样，使用标签h1~h6，但bootstrap中还添加了类“.h1~.h6”来为内联元素添加标题样式。同时标题中还可以添加<small>标签或者是.small类来在标题中添加副标题。

标题尺寸

h1(36px) h2(30px) h3(24px) h4(18px) h5(14px) h6(12px)

行高

1.1：即为文字大小的1.1倍

margin值

h1~h3: margin-top:20px; margin-bottom:10px;
h3~h6: marhin-top,margin-bottom都为10px

文本颜色和字体

继承自父元素

副标题

  • line-height:1
  • font-weight:normal
  • color:#999
  • 内的文本字体在h1~h3内，其大小都设置为当前字号的65%；而在h4~h6内的字号都设置为当前字号的75%

文本样式：通过添加类或者用标签包围

中心内容

添加.lead类让段落突出显示

内联元素文本

  • <mark>高亮
  • <del>划线，被删除的文本
  • <s>划线，无用文本
  • <ins>下划线，额外插入的文本
  • <u>为文本添加下划线
  • <small>小号文本，父容器字体大小的 85%(标题元素设置值不一样)，或者是类.small
  • <strong>通过增加font-weigt来强调一段文字
  • <em>用斜体强调一段文字

对齐

  • text-left
  • text-right
  • text-center
  • text-justify
  • text-nowrap

大小写

  • text-lowercase
  • text-uppercase
  • text-capitalize 首字母大写

缩略语

缩略语带有title属性，外观表现为带有较浅的虚线框，鼠标hover上去会变成带问号的指针，悬停在缩略语上时会出现完整title内容

  • 基本缩略语 <abbr>
  • 首字母缩略语 .initialism减小font-size

地址

  • <address>

引用<blockquote>

  • 直接引用，在其中放置p元素
  • <footer>标明引用来源
  • <cite>标明引用来源中的名称
  • .blockquote-reverse添加类，引用呈现内容右对齐效果

列表

  • 无序列表 <ul><li>
  • 有序列表 <ol><li>
  • 无样式列表 list-unstyled
    对子元素移除了默认的list-style样式和左侧外边距
  • 内联列表 list-inline
    设置 display: inline-block; 并添加少量的内补（padding）将所有元素放置于同一行
  • 描述 <dl><dt><dd>
  • 水平排列的描述 dl-horizontal
    让dl内的短语排在一行，且标题会自动截断（text-overflow属性）变成省略形式；但在较窄的视口内，会恢复默认形式

代码

  • <code>内联代码
  • <kbd>用户通过键盘输入的内容
  • <pre>代码块
  • .pre-scrollable设置 max-height 为 350px ，并在垂直方向展示滚动条
  • <var>标记变量
  • <samp>标记程序输出的内容
