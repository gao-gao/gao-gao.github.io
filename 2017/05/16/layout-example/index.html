布局实例

发表于 2017-05-16 | 分类于 html&css
自动居中一列布局；横向两列布局（多种方式实现）；三栏中间自适应布局

自动居中一列布局

  • 处于标准文档流中
  • 块级元素
  • 设置宽度和 margin：0 auto即可（常用方法，其他方法参考居中问题）

    设置浮动或绝对定位则会使自动居中auto的设置失效，因为其会脱离文档流。

浮动的横向两(多)列布局

  • float使纵向排列的块级元素横向排列
  • 通过margin设置列之间的间距

绝对定位的横向两列布局

常用于一列定宽，另一列自适应

  • relative 父元素相对定位
  • absolute 子元素绝对定位，定宽元素设置宽度，自适应元素不设置宽度

三栏中间自适应布局

两列定宽，中间自适应

方法一：绝对定位

  • 三栏父元素设置相对定位
  • 剩下三栏设置绝对定位，定宽设置宽度，自适应不设置宽度。

方法二：圣杯

  • 三栏父元素设置两边padding，留出两栏定宽部分

或者

后者学习外边距负值的意义即可，实际开发中不使用（浏览器缩小到一定范围，左右两边栏会被挤到下一行）

  • 利用绝对定位，设置两栏定宽部分

    方法三：双翼

  • 中间一栏设置margin，留出两栏定宽部分
  • 利用绝对定位，设置两栏定宽部分

    方法四：纯float方法

    左右两侧固定宽度优先渲染，中间部分插入其中间
