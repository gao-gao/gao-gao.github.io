盒子3D模型；两种盒子模型：box-sizing；各个参数代表的意义

盒子模型

盒子3D模型

由上到下依次分为五层：

  • border 边框
  • content + padding 内容与内边距
  • background-image 背景图片
  • background-color 背景颜色
  • margin 外边距

盒子模型是网页布局的基础

两种模型：box-sizing

未设置默认为content-box

  • content-box：宽高值针对内容区域，扩展边距和边框让盒子变大,不会改变内容区域大小
    content-box = 内容盒子宽高 + 填充（Padding）+ 边框宽（border-width）

  • border-box：宽高值针对border及以内区域，扩展边距和宽高，让内容变小，盒子大小不变
    border-box = 内容盒子宽高

内边距 padding

边框 border

  • 圆角：border-radius
  • 阴影：box-shadow
  • 轮廓：outline

外边距 margin

  • margin合并：
    垂直方向上，毗邻元素外间距（margin）会合并，取相对较大的值。父元素与第一个和最后一个子元素的外间距也可合并。
    水平方向上，相邻的元素，他们的水平间距是相邻外边距之和
