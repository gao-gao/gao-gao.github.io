图片css设置

发表于 2017-05-21 | 分类于 html&css
响应式图片设置，图片形状设置，图标设置

图片

通过修改图片容器大小来控制图片大小，而不是直接修改图片的CSS

响应式图片

  • img-responsive类为图片设置max-width: 100%; height: auto;display: block;
  • center-block类让上类图片居中

图片形状

  • img-rounded 圆角图片
  • img-circle 圆形图片
  • img-thumbnail 缩略图片

图标

通过@font-face属性加载了字体：在Bootstrap框架中是通过给元素添加“glyphicon”类名来实现，然后通过伪元素“:before”的“content”属性调取对应的icon编码。

  • 使用图标：在任何内联元素上添加类glyphicon和相应的图标的类名（以glyphicon加名称组合而成）
